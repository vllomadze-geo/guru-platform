# GURU Platform v1.1.10

## Что изменено

Версия v1.1.10 добавляет безопасное управление жизненным циклом проекта на стартовой странице выбора проектов.

## Добавлено

- компактное меню `⋯` на карточке проекта;
- пункты меню:
  - открыть проект;
  - редактировать паспорт;
  - экспорт диагностики проекта;
  - архивировать;
  - удалить проект;
- фильтр проектов: `Активные / Архив`;
- мягкое действие `Архивировать`, проект скрывается с главного экрана, но данные сохраняются;
- восстановление проекта из архива;
- защищённое удаление проекта через модальное окно;
- перед удалением можно экспортировать диагностику проекта;
- кнопка `Удалить навсегда` активируется только после точного ввода названия проекта.

## Что не менялось

- Supabase-подключение оставлено;
- Яндекс-блок оставлен;
- Gate 1–7 не пересобирались;
- правки v1.1.1–v1.1.9 сохранены;
- кнопка удаления не добавлена прямо на карточку как видимое опасное действие.

## Важно

В текущей HTML/MVP-версии удаление очищает проект и workspace из локального хранилища браузера. Полное каскадное удаление из Supabase по `project_id` нужно подключить отдельным backend-этапом, когда платформа перейдёт на полноценное хранение всех проектов в Supabase.
