# ГУРУ, маркетинговая операционная платформа, v0.12

Версия v0.12 продолжает очистку и системный рефакторинг платформы без добавления новых окон и разделов.

## Что изменено

1. Все Gate используют единый визуальный стандарт: таблица, одинаковые статусы, одинаковые поля и вложенность.
2. Для всех обычных блоков добавлена рабочая фиксация по инструкции блока.
3. Инструкция блока разбирается на отдельные рабочие элементы:
   - инструмент;
   - сервисная ссылка;
   - ссылка или отчёт;
   - статус проверки;
   - результат;
   - короткий комментарий.
4. Google-инструменты получают такое же рабочее пространство, как Яндекс-инструменты.
5. Сервисные ссылки отображаются внутри соответствующих блоков, а не отдельным разделом.
6. Глобальная база ссылок работает внутри блоков: введённые URL сохраняются и предлагаются далее.
7. Новые отдельные окна, разделы и пункты меню не добавлялись.
8. Данные продолжают работать по принципу снежного кома: один раз введённые данные используются в связанных местах.

## Состав файлов

- `index.html`, основной HTML
- `styles.css`, визуальная система
- `app.js`, логика платформы
- `seed-data.js`, стартовая структура из CSV
- `data/original-guru.csv`, исходный CSV

## Обновление через GitHub

Загрузить содержимое папки в репозиторий `guru-platform` через `Add file → Upload files` и сделать commit:

```text
Update to v0.12: refactor all gates by instruction logic
```

Vercel обновит сайт автоматически.
