# ГУРУ, MVP платформы v0.24

## Что изменено в v0.24

Обновлён блок **Gate 1 → Аудит сайта → SSL / HTTPS**.

### Сделано

1. Блок SSL / HTTPS пересобран как компактный site-level блок.
2. Убраны лишние сущности:
   - Яндекс Вебмастер;
   - Google Search Console;
   - Яндекс Директ;
   - Google Ads;
   - Яндекс Метрика;
   - Google Analytics;
   - CRM;
   - формы;
   - коллтрекинг;
   - рабочее пространство инструментов;
   - таблица «ссылка / источник / статус / комментарий».
3. Оставлены только три рабочих поля:
   - проверяемый домен;
   - статус SSL;
   - доказательство / результат.
4. Статус блока считается автоматически:
   - домен не указан → Не начато;
   - домен указан, но результата нет → В работе;
   - SSL ОК + доказательство есть → Готово;
   - сертификат скоро истекает → Требует внимания;
   - ошибка SSL / HTTPS → Проблема.

## Формула блока

Домен → SSL-статус → доказательство → автоматический статус.
