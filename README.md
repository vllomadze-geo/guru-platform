# ГУРУ, MVP платформы v0.19

## Что изменено в v0.19

Контекстно очищен блок `Gate 1 → Аудит сайта → Редиректы / Проверка ответа сервера`.

### Исправлено

1. Убраны из блока нерелевантные инструменты:
   - Яндекс Директ;
   - Google Ads;
   - Яндекс Метрика;
   - Google Analytics;
   - CRM;
   - Формы;
   - Коллтрекинг;
   - рабочее пространство инструментов;
   - карточки Яндекс / Google;
   - таблица `ссылка / источник / статус / комментарий`;
   - ручной общий статус внутри тела блока.
2. Оставлена минимальная структура:
   - `Проверяемый URL`;
   - `Ответ сервера`;
   - `Результат проверки`.
3. Статус блока рассчитывается автоматически:
   - URL пустой → `Не начато`;
   - URL указан, результата нет → `В работе`;
   - 200 или 301 → 200 с корректным финальным URL → `Готово`;
   - 302, длинная цепочка, http/https или www/non-www конфликт → `Требует внимания`;
   - 404 / 403 / 500 / циклический редирект → `Проблема`.
4. Инструкция остаётся свернутой по умолчанию.
5. Блок занимает компактную ширину на десктопе и 100% на мобильной версии.
