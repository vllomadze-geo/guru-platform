# УНИВЕРСАЛ / ГУРУ — MVP онлайн-платформы v0.3


Версия: `v0.2`

Это рабочий MVP маркетинговой операционной платформы на основе CSV-чеклиста.

## Что уже работает

1. Стартовая страница выбора проекта.
2. Круглые иконки проектов по логике выбора аккаунта в Google Chrome.
3. Название, описание и ссылка проекта на стартовой странице.
4. Клик по проекту открывает рабочий интерфейс выбранного проекта.
5. Кнопка `+` создаёт новый проект.
6. У каждого проекта отдельное локальное сохранение в браузере.
7. Gate 0–7 отображаются в левом меню.
8. Карточки блоков отображаются из CSV.
9. Табличный режим и карточный режим.
10. Статусы блоков: не начато, в работе, готово, проверить.
11. Доказательства, страницы и комментарии сохраняются.
12. Метрики можно вводить вручную.
13. Есть простой график расходов и лидов.
14. Есть схема прохождения Gate.
15. Есть импорт CSV и экспорт CSV.

## Важное ограничение

Данные пока сохраняются в браузере через `localStorage`.

Это значит:

- на другом устройстве данные не появятся;
- при очистке браузера данные могут пропасть;
- это ещё не полноценная база данных.

Следующий крупный этап: подключение Supabase.

## Файлы проекта

- `index.html` — структура интерфейса.
- `styles.css` — визуальный стиль.
- `app.js` — логика платформы.
- `seed-data.js` — стартовая структура Gate 0–7.
- `data/original-guru.csv` — исходный CSV.

## Как обновлять онлайн-версию

1. Загрузить изменённые файлы в GitHub.
2. Сделать commit.
3. Vercel автоматически обновит сайт.

## Что добавлено в v0.3

1. Графа «Доказательство» больше не является одной общей ячейкой.
2. Смысловые пункты внутри доказательства автоматически становятся отдельными полями.
3. Значения сохраняются как единые данные проекта.
4. Если один и тот же заголовок используется в разных блоках, значение подтягивается во все эти блоки.
5. Добавлен отдельный раздел «Доказательства» для редактирования всех единых полей проекта.
6. Экспорт CSV теперь включает отдельный блок `SHARED_EVIDENCE`.
