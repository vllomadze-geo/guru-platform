# ГУРУ, MVP платформы v0.25

## Что изменено в v0.25

Пересобран блок `Gate 1 → Спрос: семантика, кластеризация, намерения`.

Теперь это не склад семантики, а единый маршрут подготовки кампании к запуску:

`Рамки → язык аудитории → намерения → группы → экономика → объявления → запуск`.

## Реализовано

1. Один общий маршрут «Спрос».
2. Выбор рекламной системы: Яндекс / Google / Яндекс + Google.
3. Выбор типа кампании: Поиск / РСЯ / Search / Display.
4. Верхний блок рамок продвижения.
5. 7 пошаговых секций маршрута.
6. Финальная таблица запуска.
7. Отдельная логика для Search keywords и Display audiences.
8. Автоматические проверки проблем.
9. Автоматический статус блока.

## Принцип

Не собрать семантику ради семантики, а подготовить кампанию к запуску.
