# ГУРУ, MVP платформы v0.16

## Что изменено в v0.16

Контекстно переработан блок:

`Gate 1 → Аналитика → Аудит сайта → Robots.txt`

### Исправлено

1. Один блок `Robots.txt` вместо раздробленной логики `Robots.txt Яндекс`.
2. Один URL robots.txt.
3. Переключатели систем: `Яндекс` и `Google`.
4. Для каждой выбранной системы показывается компактная карточка:
   - статус проверки;
   - доказательство.
5. Общий статус блока считается только по выбранным системам.
6. Отключённая система не влияет на готовность блока.

## Логика статуса

- Нет URL и не выбраны системы, `Не начато`.
- URL есть, но выбранные системы заполнены не полностью, `В работе`.
- По всем выбранным системам указан статус и доказательство, `Готово`.
