# GURU Platform v0.20

Контекстная версия аудита сайта.

## Главное изменение

Gate 1 → Аудит сайта перестроен по принципу: страница является центром управления SEO, скоростью, изображениями и конверсией.

## Что изменено

- Meta Robots, CWV PageSpeed Insights, изображения, Snippet и Финальный CTA встроены внутрь карточек страниц.
- Отдельные блоки Meta Robots, CWV и Изображения работают как сводки проблем по страницам, без ручного заполнения.
- Site-level проверки остаются отдельными компактными блоками: Robots.txt, Sitemap.xml, Редиректы / ответ сервера, SSL, Cookie-banner, Footer, согласия в формах.
- Статусы страниц считаются автоматически по URL, SEO-полям, Meta Robots, CWV, изображениям и CTA.
- Сводки показывают агрегированный результат по страницам и позволяют перейти к конкретной странице.

## Обновление

Загрузить содержимое папки в GitHub поверх текущей версии и дождаться деплоя Vercel.
