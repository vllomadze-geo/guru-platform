# GURU Platform v1.1.4

Изменён только блок Gate 0 / 0. Точка отсчёта → «Текущая семантика и поисковый фокус».

## Сохранено

- Supabase оставлен.
- Яндекс оставлен.
- Gate 1–7 не тронуты.
- Предыдущие правки v1.1.1, v1.1.2 и v1.1.3 сохранены.

## Изменено

Блок «Текущая семантика и поисковый фокус» возвращён к стартовой паспортной фиксации поискового направления:

- Основные ключевые слова.
- Приоритетные кластеры.
- Брендовые запросы.
- Нерелевантные запросы.
- Итоговый поисковый фокус.
- Доказательство.

## Убрано из этого блока

- Отдельные рабочие строки Яндекс Директ и Яндекс Wordstat.
- Большая таблица инструментов.
- Повторяющиеся поля ссылка / отчёт / статус / результат / комментарий.
- Ручной статус каждой строки.
- Отдельная строка «Итоговый результат блока».

## Глобальные данные

Поля заложены как глобальный поисковый фокус проекта:

- project.searchMainKeywords
- project.searchPriorityClusters
- project.searchBrandQueries
- project.searchIrrelevantQueries
- project.searchFocusSummary
- project.searchFocusProof

Полная передача в конкретные поля Gate 1, SEO, страниц, рекламы и отчётов будет подключаться отдельным этапом.
