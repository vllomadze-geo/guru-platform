# ГУРУ, MVP платформы v0.15

## Что изменено в v0.15

Контекстно очищен блок **Gate 1 → Аналитика → Аудит сайта → Robots.txt Яндекс**.

### Исправлено

1. Из блока убраны инструменты, которые не относятся к проверке robots.txt через Яндекс:
   - Яндекс Директ;
   - Google Ads;
   - Яндекс Метрика;
   - Google Analytics;
   - CRM;
   - Формы;
   - Коллтрекинг;
   - Google Search Console.

2. В блоке оставлен только рабочий контур Яндекс Вебмастера:
   - URL сайта;
   - URL robots.txt;
   - инструмент проверки: Яндекс Вебмастер;
   - статус файла: найден / не найден;
   - статус анализа: корректен / есть ошибки;
   - важные страницы открыты: да / нет;
   - служебные страницы закрыты: да / нет;
   - Sitemap указан в robots.txt: да / нет;
   - ссылка на проверку / скрин / отчёт;
   - маленький комментарий только при проблеме.

3. Логика блока приведена к формуле:

`URL → Яндекс Вебмастер → статус файла → результат анализа → доказательство`
