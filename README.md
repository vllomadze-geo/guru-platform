# ГУРУ, MVP платформы v0.18

## Что изменено в v0.18

Контекстно очищен блок `Gate 1 → Аудит сайта → Sitemap.xml`.

### Исправлено

1. Блок `Sitemap.xml` больше не проверяет рекламную, аналитическую и CRM-инфраструктуру.
2. Убраны из блока:
   - Яндекс Директ;
   - Google Ads;
   - Яндекс Метрика;
   - Google Analytics;
   - CRM;
   - Формы;
   - Коллтрекинг;
   - рабочее пространство включённых инструментов;
   - отдельная таблица `ссылка / источник / статус / комментарий`.
3. Оставлена минимальная структура:
   - один `URL sitemap.xml`;
   - переключатели `Яндекс / Google`;
   - карточка выбранной системы;
   - `Статус проверки`;
   - `Доказательство / результат`.
4. Невыбранная система скрывается.
5. Инструкция по умолчанию остаётся свернутой.
6. Статус блока рассчитывается автоматически только по выбранным системам.

### Логика статуса

- URL sitemap.xml пустой → `Не начато`.
- URL указан, но системы не выбраны → `В работе`.
- Система выбрана, но статус не указан → `В работе`.
- Выбранная система = `ОК` и доказательство есть → `Готово`.
- Яндекс и Google выбраны, оба = `ОК` и оба с доказательством → `Готово`.
- В любой выбранной системе стоит `Ошибка` → `Проблема`.
