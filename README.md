# ГУРУ, MVP платформы v0.11

## Что изменено в v0.11

1. Убраны лишние пункты бокового меню: `Ссылки проекта`, `Инструменты`, `Доказательства`.
2. Глобальная база ссылок сохранена как внутренняя автоматизация: URL, введённые в блоках, продолжают предлагаться в других блоках.
3. Управление источниками данных встроено внутрь рабочих блоков Gate 1, без отдельного окна.
4. Структурированные доказательства остаются внутри рабочих блоков, без отдельного раздела.
5. Закреплено правило: новые разделы, окна, пункты меню и сущности не добавляются без прямого запроса пользователя.
6. Обновлена версия интерфейса до v0.11.

## Как обновить GitHub

Загрузить содержимое папки `guru_platform_v10` в репозиторий `guru-platform` через `Add file → Upload files`.

Commit:

`Update to v0.11: simplify navigation and embed automation`


## v0.11

- Удалён режим «Карточки», оставлен только режим «Таблица».
- Удалена общая графа «Комментарий» из рабочих таблиц.
- Статусы усилены цветом по смыслу: готово, работа, проблема, нейтрально.
- Gate 1 получил более явную иерархию: путь, активный подблок, активный рабочий блок и левая цветовая полоса.
- Google Search Console, Google Ads и другие включённые инструменты теперь открывают рабочие пространства внутри блоков, а не остаются простыми чекбоксами.
- SEO-плашки убраны из технических строк ссылок, SEO-нюансы остаются внутри структуры конкретных страниц.
- Логика ссылок и инструментов остаётся встроенной в существующие Gate и блоки, без новых окон.
