# УНИВЕРСАЛ / ГУРУ, MVP онлайн-платформы v0.4

Это рабочая версия платформы на базе CSV-структуры ГУРУ.

## Что работает

1. Стартовая страница выбора проекта.
2. Создание нового проекта через кнопку `+`.
3. Данные проекта синхронизируются с разделом `0. Точка отсчёта`, блок `Паспорт проекта`.
4. В рабочем интерфейсе отображается название текущего проекта.
5. Статус блока рассчитывается автоматически:
   - пусто, `Не начато`;
   - текст есть, но нет точки в конце, `В работе`;
   - все обязательные поля заполнены и текстовые поля заканчиваются точкой, `Готово`.
6. Графа `Доказательство` структурирована на отдельные поля.
7. Одинаковые поля доказательств синхронизируются по всей системе проекта.
8. В блоках с инструментами и платформами есть отдельные строки со статусом `Реализовано / Не реализовано`.
9. Блок `Проблемы и ограничения на старте` автоматически собирает сводку `Да / Нет`.
10. Есть экспорт CSV.
11. Есть экспорт в PDF через печать браузера, в отчёт попадают только:
    - название блока;
    - статус;
    - структурированное доказательство;
    - комментарий.

## Важно

Данные пока сохраняются в браузере через `localStorage`. Для полноценной совместной работы следующим этапом нужно подключить Supabase.

## Файлы

- `index.html`, структура интерфейса.
- `styles.css`, внешний вид.
- `app.js`, логика платформы.
- `seed-data.js`, стартовые данные из CSV.
- `data/original-guru.csv`, исходный CSV.
