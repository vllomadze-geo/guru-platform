# ГУРУ, MVP платформы v0.9

Рабочий прототип маркетинговой операционной платформы.

## Что есть в v0.9

- Стартовая страница выбора проекта.
- Создание нового проекта через кнопку `+`.
- Gate 0–7.
- Структурированные доказательства.
- Синхронизация данных проекта.
- Автоматические статусы блоков.
- Паспорт проекта с состоянием до и после работ.
- Экспорт CSV и PDF по текущему Gate.
- Gate 1 с раскрывающимися подблоками.
- Типизированные поля Gate 1: ссылки, сравнительные показатели, структура страниц.
- Глобальная база ссылок проекта.
- SEO-индикатор для каждой ссылки.
- Выбор источника данных в блоках Gate 1.
- Управление инструментами проекта: включён / отключён.
- Отключённые инструменты не влияют на готовность блоков.

## Важное ограничение

Данные пока сохраняются в браузере через localStorage. Для полноценной командной работы нужен Supabase.

## Как обновлять на GitHub

Загрузить содержимое папки в репозиторий `guru-platform` через:

`Add file → Upload files`

Commit:

`Update to v0.9: project link bank and tool settings`
