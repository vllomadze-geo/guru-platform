# ГУРУ Platform MVP v0.14

Контекстная версия ГУРУ: один блок, одна понятная задача, только нужные поля.

## Что изменено

- Убрана логика универсального дублирования инструментов во всех блоках.
- Каждый блок показывает поля по своему контексту: страница, аналитика, стратегия, реализация, задача.
- Блоки страниц строятся по структуре страницы, а не по списку инструментов.
- Для блока «Главная» добавлены смысловые группы: герой-экран, навигация по сегментам, о компании, кейсы, процесс, выгоды, отзывы, сниппет и финальный CTA.
- Отдельные блоки «Сниппет» и «Финальный CTA» скрыты из Gate 1, их логика встроена внутрь страниц.
- Gate 2, Gate 3 и Gate 4 структурированы по раскрывающимся смысловым подгруппам.
- Данные из проекта и предыдущих Gate подтягиваются только там, где они логически нужны.
- Новые окна и новые пункты меню не добавлены.
