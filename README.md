# ГУРУ, MVP платформы v0.17

## Что изменено в v0.17

Контекстно пересобран блок `Gate 1 → Аудит сайта → Robots.txt`.

### Исправлено

1. Блок собран по сетке из 3 уровней.
2. Верхний уровень: один `URL robots.txt`, переключатели `Яндекс / Google`, статус блока в заголовке.
3. Средний уровень: компактные карточки выбранных систем.
4. Внутри карточки только два поля: `Статус проверки` и `Доказательство`.
5. Невыбранные системы скрываются.
6. Убран общий выпадающий статус внутри тела блока.
7. Убраны дублирующие заголовки и лишние плашки.
8. Статус блока считается автоматически только по выбранным системам.

### Логика статуса

- URL пустой → `Не начато`.
- URL есть, но системы не выбраны → `В работе`.
- Система выбрана, но статус не указан → `В работе`.
- Выбранная система имеет `ОК` и доказательство → `Готово`.
- Яндекс и Google выбраны, оба `ОК` и оба с доказательством → `Готово`.
- В выбранной системе `Ошибка` → `Проблема`.
