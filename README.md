# УНИВЕРСАЛ / ГУРУ, MVP онлайн-платформы v0.7

Это рабочая версия платформы на базе CSV-структуры ГУРУ.

## Что работает

1. Стартовая страница выбора проекта.
2. Создание нового проекта через кнопку `+`.
3. Данные проекта синхронизируются с разделом `0. Точка отсчёта`, блок `Паспорт проекта`.
4. В рабочем интерфейсе отображается название текущего проекта.
5. Статус блока рассчитывается автоматически.
6. Графа `Доказательство` структурирована на отдельные поля.
7. Одинаковые поля доказательств синхронизируются по всей системе проекта.
8. В блоках с инструментами и платформами есть отдельные строки со статусом `Реализовано / Не реализовано`.
9. Блок `Проблемы и ограничения на старте` автоматически собирает сводку `Да / Нет`.
10. Есть экспорт CSV.
11. Есть экспорт в PDF по текущему Gate.

## v0.7

- Gate 1 `Аналитика` больше не отображается как одна длинная линейная таблица.
- Внутри Gate 1 добавлены 4 раскрывающихся подблока:
  1. `Аудит сайта`;
  2. `Спрос: семантика, кластеризация, намерения`;
  3. `Боль → JTBD → офер`;
  4. `Юнит-экономика: целевые CPA/DRR, AOV/LTV, маржинальность`.
- В заголовке каждого подблока отображаются статус и прогресс, например `3 из 12 блоков готово`.
- Каждый рабочий блок внутри подблока тоже открывается и закрывается отдельно.
- В закрытом виде рабочий блок показывает название, статус, краткий результат и комментарий.
- В открытом виде показываются инструкция, поля заполнения, доказательство, размещение и комментарий.
- Состояние раскрытия подблоков и рабочих блоков сохраняется внутри проекта.
- В подблок `Юнит-экономика` добавлены рабочие блоки: `Целевой CPA`, `DRR`, `AOV`, `LTV`, `Маржинальность`, `Ограничения по экономике`.

## Важно

Данные пока сохраняются в браузере через `localStorage`. Для полноценной совместной работы следующим этапом нужно подключить Supabase.
