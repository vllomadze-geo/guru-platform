# GURU Platform v1.1.7 — Экспорт диагностики

## Что добавлено

Добавлен инструмент **GURU Debug Export / Экспорт диагностики**.

## Кнопки

- В панели Gate: `Экспорт диагностики Gate`.
- В паспортных блоках Gate 0: `Экспорт диагностики блока`.

## Что экспортируется

Файл `guru-debug_...json` содержит:

- gate;
- blockName;
- blockId;
- status;
- fields;
- evidenceFields;
- validationErrors;
- globalDataLinks;
- syncConflicts;
- uiState;
- technical data;
- consoleErrors;
- notes.

## Что не трогалось

- Supabase оставлен.
- Яндекс оставлен.
- Gate 1–7 не пересобирались.
- Правки Gate 0 v1.1.1–v1.1.6 сохранены.

## Как использовать

1. Открой нужный Gate или блок.
2. Нажми `Экспорт диагностики блока` или `Экспорт диагностики Gate`.
3. Введи короткое описание проблемы.
4. Скачай JSON-файл и пришли его вместе со скриншотом.

Формула: скриншот показывает внешний вид, debug export показывает причину проблемы.
