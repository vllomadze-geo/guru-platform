# УНИВЕРСАЛ / ГУРУ, MVP онлайн-платформы v0.5

Это рабочая версия платформы на базе CSV-структуры ГУРУ.

## Что работает

1. Стартовая страница выбора проекта.
2. Создание нового проекта через кнопку `+`.
3. Данные проекта синхронизируются с разделом `0. Точка отсчёта`, блок `Паспорт проекта`.
4. В рабочем интерфейсе отображается название текущего проекта.
5. Сводные карточки `Gate`, `Блоки`, `Готово`, `Метрики` отображаются только в разделе `Паспорт проекта`.
6. В паспорте проекта есть две вертикали: `До начала работ` и `После завершения работ`.
7. Статус блока рассчитывается автоматически.
8. Графа `Доказательство` структурирована на отдельные поля.
9. Одинаковые поля доказательств синхронизируются по всей системе проекта.
10. Служебные подписи синхронизации убраны из интерфейса.
11. В блоках с инструментами и платформами есть отдельные строки со статусом `Реализовано / Не реализовано`.
12. Блок `Проблемы и ограничения на старте` автоматически собирает сводку `Да / Нет`.
13. Блок `Текущие результаты` содержит вертикальную таблицу показателей: значение, период, комментарий.
14. Показатель не считается готовым, если есть значение без периода.
15. Есть экспорт CSV.
16. Экспорт PDF выгружает только текущий Gate, в отчёт попадают только:
    - название блока;
    - статус;
    - структурированное доказательство;
    - комментарий.

## Важно

Данные пока сохраняются в браузере через `localStorage`. Для полноценной совместной работы следующим этапом нужно подключить Supabase.

## Файлы

- `index.html`, структура интерфейса.
- `styles.css`, внешний вид.
- `app.js`, логика платформы.
- `seed-data.js`, стартовые данные из CSV.
- `data/original-guru.csv`, исходный CSV.
