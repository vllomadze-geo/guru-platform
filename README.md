# ГУРУ, MVP платформы v0.26

## Что изменено в v0.26

Контекстно пересобран блок:

`Gate 1 → Боль → JTBD → офер`

### Исправлено

1. Убран длинный список из множества отдельных карточек.
2. Блок собран в один маршрут из 7 accordion-разделов:
   - Поисковый спрос
   - Конкурентное предложение
   - Боли и причины спроса
   - JTBD
   - ЦА, сегменты и персоны
   - Офер
   - Коллаборационный потенциал
3. Вверху блока добавлены поля:
   - Продукт / услуга
   - Посадочная / проект
   - Главный сегмент
   - Финальный офер
4. В каждом разделе есть:
   - ориентир
   - 2–4 рабочих поля
   - стандарт готовности
   - доказательство
   - автоматический статус
5. Общий статус блока считается автоматически.
6. Сохранена логика без новых окон и дополнительных пунктов меню.

## Формула блока

Спрос → конкуренты → боли → JTBD → сегменты → офер → партнёрское усиление.
