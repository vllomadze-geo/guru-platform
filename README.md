# ГУРУ, MVP платформы v0.22

## Что изменено в v0.22

Карточки страниц в Gate 1 → Аудит сайта перестроены из универсальной анкеты в маршрутные шаблоны по типам страниц.

### Реализовано

1. Для каждого типа страницы создан отдельный шаблон маршрута:
   - Главная;
   - Лендинг;
   - Страница услуги;
   - Список / категория;
   - Карточка товара;
   - Статья блога;
   - О нас;
   - Контакты;
   - Доставка / гарантии;
   - Политика;
   - 404;
   - Thank You Page.

2. Карточка страницы теперь работает по формуле:

```
ориентир → секции страницы → чек-пункты → стандарт → доказательство → статус
```

3. Внутри каждой секции есть:
   - задача секции;
   - чек-пункты;
   - стандарт готовности;
   - поля результата;
   - доказательство;
   - автоматический статус секции.

4. По умолчанию открывается первая незаполненная секция.

5. Внизу карточки страницы остаётся компактный технический контроль:
   - Meta Robots;
   - CWV;
   - Изображения;
   - доказательство.

6. Статус страницы считается автоматически.
