# ГУРУ, MVP платформы v0.23

## Что изменено в v0.23

Обновлены блоки Gate 1 → Аудит сайта → юридический и доверительный контроль.

### Сделано

1. **Cookie-баннер** пересобран как компактный legal/trust-блок:
   - URL проверки;
   - статус cookie-баннера;
   - доказательство / результат;
   - чек-пункты: баннер показывается, текст понятный, ссылка на политику, не перекрывает ключевое действие.

2. **Футер: Политика + Реквизиты** пересобран как компактный legal/trust-блок:
   - URL проверки;
   - статус футера;
   - доказательство / результат;
   - чек-пункты: политика, реквизиты, контакты, ссылки открываются.

3. **Согласие в формах** больше не ручная форма по всему сайту, а сводка проблем по страницам.

4. Внутри карточек страниц добавлен контроль формы:
   - форма есть / формы нет / не проверено;
   - согласие в форме: не проверено / ОК / ошибка;
   - доказательство;
   - причина проблемы.

5. Из этих блоков убраны рекламные, SEO, аналитические и CRM-инструменты, которые не относятся к юридико-доверительной проверке.

## Формула блока

Объект проверки → короткий стандарт → результат → доказательство → статус.
