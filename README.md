# УНИВЕРСАЛ / ГУРУ — MVP онлайн-платформы

Это первая рабочая версия платформы из CSV-чеклиста.

## Что уже работает

1. Открывается как обычный сайт через `index.html`.
2. Загружает структуру Gate 0–7 из исходного CSV.
3. Показывает блоки как карточки или таблицу.
4. Даёт менять статус блока.
5. Даёт добавлять доказательства, страницы и комментарии.
6. Автоматически сохраняет данные в браузере через `localStorage`.
7. Позволяет импортировать CSV.
8. Позволяет экспортировать CSV.
9. Показывает простую схему Gate и график прогресса.
10. Даёт отдельный раздел для метрик: показы, клики, расход, лиды, CTR, CPA.

## Как открыть

Самый простой способ:

1. Распакуйте архив.
2. Откройте файл `index.html` двойным кликом.
3. Платформа откроется в браузере.

## Важно

Эта версия сохраняет данные в браузере конкретного компьютера. Это удобно для первого прототипа, но это ещё не полноценная база данных.

Для настоящей онлайн-платформы следующим этапом нужно подключить:

1. GitHub — хранение кода.
2. Vercel — публикация платформы в интернете.
3. Supabase — база данных для сохранения информации не только в браузере.

## Структура файлов

- `index.html` — страница платформы.
- `styles.css` — внешний вид.
- `app.js` — логика платформы.
- `seed-data.js` — структура Gate и блоков, собранная из CSV.
- `data/original-guru.csv` — исходный CSV-файл.
