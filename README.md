# GURU Platform v1.1.6

Точечное обновление Gate 0: единый компактный интерфейс для 5 базовых паспортных блоков проекта.

## Что сохранено

- Supabase оставлен.
- Яндекс оставлен.
- Gate 1–7 не изменялись.
- Все правки v1.1.1–v1.1.5 сохранены.

## Что изменено

Обновлен интерфейс пяти блоков Gate 0:

1. Продукт, сегмент и задача клиента.
2. Психологическая декомпозиция оффера.
3. Текущее позиционирование и УТП.
4. Текущие офферы и CTA.
5. Текущая семантика и поисковый фокус.

## Новый стандарт блока

Каждый блок теперь строится по одной архитектуре:

1. Короткий ориентир.
2. Инструкция скрыта по умолчанию.
3. Автоконтекст свернут.
4. Главные рабочие поля видимые и крупные.
5. Отдельное место для ключевых слов / смыслов / тегов.
6. Одно поле доказательства.
7. Индикатор передачи данных дальше.
8. Автоматический статус.

## Главный принцип

Не обучать в рабочем поле. Рабочее поле фиксирует результат.

Формула интерфейса:

смысл → ключевые слова → доказательство → передача дальше → статус.
