# ГУРУ, MVP платформы v0.29

## Что изменено в v0.29

Интегрирован уже реализованный HTML-инструмент рекламной отчётности внутрь **Gate 5 → Оценка и оптимизация, постоянная петля**.

Инструмент больше не живёт как отдельное приложение:

- убрана тёмная тема;
- убран отдельный header `ГУРУ · рекламный пульт`;
- убрано верхнее меню `Настройка / Ввод / Реклама / Связка / Финансы`;
- разделы инструмента встроены как подблоки Gate 5;
- используется светлая сетка, карточки, кнопки, поля и статусы ГУРУ;
- не создаётся отдельный режим или отдельное окно.

## Структура Gate 5

1. **Настройка отчётности**
   - структура кампаний;
   - группы;
   - объявления;
   - импорт структуры XLSX / CSV.

2. **Ввод данных**
   - импорт отчётов XLSX / CSV;
   - цели;
   - история целей.

3. **Рекламная оценка**
   - кампании;
   - расход;
   - показы;
   - клики;
   - лиды;
   - CPA;
   - проблемные запросы.

4. **Связка с бизнесом**
   - лиды;
   - заказы;
   - выручка;
   - конверсия лид → заказ.

5. **Финансовая оценка**
   - CPA;
   - CAC;
   - ROAS;
   - DRR;
   - ROI;
   - решение: масштабировать / оптимизировать / остановить.

## Сохранён функционал

- импорт XLSX / CSV;
- настройка структуры кампаний;
- ввод целей;
- анализ рекламы;
- связка лиды → заказы → выручка;
- финансовая оценка CPA, CAC, ROAS, DRR, ROI;
- локальное сохранение данных внутри проекта;
- автоматические статусы и выводы.

## Главная логика Gate 5

`данные → цель → реклама → лиды → заказы → финансы → решение → следующая оптимизация`
